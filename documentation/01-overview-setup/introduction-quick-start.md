# Introduction & Quick Start

MYJUNGLE is a persistent code intelligence platform designed for AI coding agents. It provides a robust infrastructure to index repositories once, maintain freshness through incremental updates, and expose high-quality code context via read-oriented Model Context Protocol (MCP) tools. The platform supports multiple projects from its inception, utilizing a microservices architecture to handle parsing, embedding, and storage efficiently.

The system is built to facilitate seamless integration between AI agents and codebase knowledge. By providing structured access to symbols, dependencies, and semantic search, MYJUNGLE enables agents to retrieve accurate context without the overhead of manual repository management.
